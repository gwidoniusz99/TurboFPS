# TurboFPS
##A lightweight, easy-to-use Windows batch tool created for gamers. It helps boost FPS, reduce stuttering, and increase overall system responsiveness without installing heavy software.

## Main Features:

- Full Optimization – comprehensive tuning including cleanup, disabling telemetry & unnecessary services, game priority tweaks, and performance settings
- Temporary Files Cleanup
- Error Scanning (DISM, SFC, CHKDSK, etc.)
- Game Optimizer – add games and apply/remove high priority & GPU optimizations
- Overclocking
- Driver & software updates checker
Requires administrator privileges to run, it is recommended to create a restore point.
